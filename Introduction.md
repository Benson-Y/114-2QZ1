# 目前可分析的資料、位置與說明
上一學期的巨量資料分析與應用的課程我們有用2014~2021年的車禍各車種統計數據做練習，主要是將原始的資料作拆解、分組、組合，形成一個完整的報表，這份資料我們使用R語言寫的，而我們如果要對資料進行整理，就需要用到三個套件，分別是tidyverse、lubriate、magrittr，這三套件視作數據分析、處裡日期時間的資料、以及做管線運算，tidyverse就是坐前面所說的數據的拆解分組組合，lubriate將資料日期格式另外拉出來處理，magrittr將程式中的指令串接，上一個程式的結果直接傳送到下一個命令。
